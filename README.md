## ✨ Features

- Two-column responsive layout
- Custom Norse Bold font
- Background image sidebar
- Styled form inputs
- Focus and validation states
- Create Account button
- Clean and modern UI

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- Custom Fonts

## 📚 What I Learned

Through this project, I learned how to:

- Structure forms using semantic HTML
- Build layouts with CSS Flexbox
- Import and use custom fonts with `@font-face`
- Style form elements and input states
- Create visually appealing user interfaces
- Organize project files for GitHub
